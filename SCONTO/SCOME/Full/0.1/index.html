Description

This ontology is an extension of the SCONTO ontology, presented via this academic paper. It imports all SCOME modules and thus gives the fullest expressivity that SCOME allows. You can use this ontology as-is, or you can construct your own ontology by importing individual SCOME modules.

See also: Supply Chain Process Ontology (SCOPRO) Supply Chain Benchmarking Ontology (SCOBE)

History Note:

Version 0.1 is an initial release that contains the full SCOME ontology
